Introdução:

Este projeto propõe o desenvolvimento de um webcrawler para extrair dados de preços de produtos do site da CEPEA (Centro de Estudos Avançados em Economia Aplicada) e enviá-los diretamente ao produtor através de um chatbot no WhatsApp. O objetivo é fornecer uma ferramenta que facilite a obtenção de informações em tempo real, ajudando produtores a tomar decisões mais rápidas e informadas sobre seus processos de compra.

O projeto foi desenvolvido na linguagem de programação Python (Python Software Foundation, 2024A.), utilizando as bibliotecas Selenium (Selenium, 2024), Flask (Flask, 2024), Requests (Kenneth, 2024).
